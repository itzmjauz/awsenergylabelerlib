============
Installation
============

At the command line::

    $ pip install awsenergylabelerlib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv awsenergylabelerlib
    $ pip install awsenergylabelerlib

Or, if you are using pipenv::

    $ pipenv install awsenergylabelerlib

Or, if you are using pipx::

    $ pipx install awsenergylabelerlib
